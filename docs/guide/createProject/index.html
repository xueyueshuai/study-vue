# 创建项目

# vue create

运行以下命令来创建一个新项目：

vue create hello-world

TIP

你会被提示选取一个 preset。你可以选默认的包含了基本的 Babel + ESLint 设置的 preset，也可以选“手动选择特性”来选取需要的特性。

# 用图形化界面

你也可以通过 vue ui 命令以图形化界面创建和管理项目：

vue ui
